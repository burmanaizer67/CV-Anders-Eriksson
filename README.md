# CV - Anders Eriksson

Ett professionellt CV i HTML-format med responsiv design och A4-utskriftsformat.

## Innehåll

- **CV_Anders_Eriksson.html** - Komplett CV med styling
- **Anders.jpg** - Profilbild

## Visa CV:et

Du kan öppna `CV_Anders_Eriksson.html` direkt i en webbläsare för att se CV:et.

## GitHub Pages

CV:et är tillgängligt live på: [https://burmanaizer67.github.io/CV-Anders-Eriksson/CV_Anders_Eriksson.html](https://burmanaizer67.github.io/CV-Anders-Eriksson/CV_Anders_Eriksson.html)

## Skriva ut till PDF

1. Öppna CV:et i en webbläsare
2. Tryck Cmd+P (Mac) eller Ctrl+P (Windows)
3. Välj "Spara som PDF"
4. Spara filen

## Funktioner

- ✅ A4-format (210mm x 297mm)
- ✅ Responsiv design
- ✅ Professionell layout
- ✅ Optimerad för utskrift
- ✅ Lugnande färgschema
